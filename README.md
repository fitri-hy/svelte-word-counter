# Svelte Word Counter

<img src="./Screenshot 2024-02-28 211804.png"/>

Word counter project idea using svelte + tailwind css framework

## Clone & Run Project
```sh
git clone https://github.com/fitri-hy/svelte-word-counter.git
```
```sh
cd svelte-word-counter
```
```sh
npm install
```
```sh
npm run dev
```

## Builder
```sh
npm run build
```
